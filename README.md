# 전통문화상품 B2C 웹사이트 '숨결'

### 🔗 [배포 링크](https://sumgyeol-25e36.firebaseapp.com/)  


<br />

## 프로젝트 소개

전통문화상품을 만드는 대한민국 명장들을 홍보하기 위한 상품 판매 및 체험 프로그램 안내 사이트  


<br />

## 개발 기간

2023.09.22 ~ 10.13 (2주)  


<br />

## 기술 스택

[ 개발 언어 ] &nbsp; React, Typescript, Javascript, HTML, CSS  
[ 라이브러리 ] &nbsp; Redux, React-router-dom, Styled-components, React-youtube, React-wavify, Swiper  
[ API ] &nbsp; Kakao 지도 API  
[ UI 디자인 ] &nbsp; Figma, Photoshop  


<br />

## 멤버 구성

UX/UI 기획 및 디자인 1명(서유민), 프론트엔드 3명(김지윤, 양수연, 최가은)
